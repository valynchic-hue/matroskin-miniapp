# matroskin-miniapp
Вместе с Матроскиным из «Простоквашино» ребята проходят 10 дней математических испытаний: каждый день — 3–4 интерактивных задания, которые помогают закрепить темы из школьной программы. Задания поданы в формате мини-историй: Матроскин даёт поручения, а ребёнок решает задачи, чтобы помочь герою.
